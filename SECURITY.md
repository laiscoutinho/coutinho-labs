# Security Policy

## Supported Versions

This project is under active development. Only the latest version is currently supported.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue.

Instead, report it directly to the maintainer:

contact@coutinholabs.com

---

## Response

We are committed to:

- Reviewing reported vulnerabilities promptly  
- Prioritizing critical security fixes  
- Maintaining transparency whenever possible  

---

## Scope

This policy applies to:

- Source code in this repository  
- Deployment configurations  
- Project dependencies  

---

Security is treated as a priority in the development of this project.
